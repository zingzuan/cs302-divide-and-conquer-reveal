Master Theorem

\[T(n) = aT(n / b) + f(n)\]

  1. If \(f(n) = O(n^{\log_b a - \epsilon})\), then \(T(n) = \Theta(n^{\log_b a})\).
  2. If \(f(n) = \Theta(n^{\log_b a})\), then \(T(n) = \Theta(n^{\log_b a}\log n)\).
  3. If \(f(n) = \Omega(n^{\log_b a + \epsilon})\), then \(T(n) = \Theta(f(n))\).

Summary

Divide and conquer algorithms

  • Design: Two steps
    • Divide the original problem into smaller sub-problems
    • Combine solutions of sub-problems to build the solution of the original problem
      • Key step!
  • Analysis
